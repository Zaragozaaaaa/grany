# Грань

Веб-приложение самопознания через истории. Пользователь рассказывает ситуацию и свою реакцию на неё, AI возвращает честное «зеркало» — инсайт о том, что это говорит о пользователе. Со временем истории складываются в портрет: поведенческие компетенции (грани CASEL), области жизни (Wheel of Life), навыки с уровнями.

## Стек

- **Фронт**: HTML + JS + SVG, хостинг Cloudflare Pages.
- **Бэкенд**: Supabase — Postgres + Edge Functions (Deno/TypeScript).
- **AI**: Anthropic API, модель по умолчанию `claude-sonnet-4-6`. Вызов только из edge-функции, ключ никогда не попадает во фронт.
- **Версии**: Git + GitHub, автодеплой на Cloudflare Pages.

## Сквозные требования

| ID | Требование |
|---|---|
| NF-1 | AI-ключ хранится только в секретах edge-функции |
| NF-2 | Каждая запись/навык привязаны к `user_id`; RLS в фазе роста |
| NF-3 | Ответ зеркала — в пределах ~10 сек |
| NF-4 | AI-функции извлечения возвращают строго валидный JSON без обёрток |
| NF-5 | Секреты и `.env` — в `.gitignore`, не коммитятся |

## Этапы разработки (MVP = Э0-Э5)

0. Каркас и деплой
1. Зеркало (живой AI-инсайт)
2. Память (хранение историй)
3. Захват граней (пассивное тегирование CASEL)
4. Портрет (визуализация SVG)
5. Навыки и уровни (лесенки)

Контрольная точка: полный MVP после Э5.

Фаза роста (вне MVP): Э6 мульти-юзер, Э7 автоматизация, Э8 матчинг.
