# Грань

Веб-приложение самопознания через истории.

## Стек

HTML + JS + SVG (Cloudflare Pages) · Supabase (Postgres + Edge Functions) · Anthropic API

## Разработка

Статический сайт, без сборки. Открой `index.html` или подними локальный сервер:

```bash
npx serve .
```

## Деплой

Автодеплой на Cloudflare Pages при пуше в `main`.
